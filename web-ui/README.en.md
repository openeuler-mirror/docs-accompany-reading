vue3 starter

### Software Architecture
Vue 3 + TypeScript + Vite + Scss + Vue Router + Pinia + Axios

### Installation

1. recommend using [pnpm](https://pnpm.io/installation)，you can also use ```npm```、```yarn```
2. install:  ```pnpm install```
3. run project: ```pnpm dev```
